About Me

A deeper look into who I am, what I do, and what drives me.

Profile

I am Emir, a software developer and Linux distribution developer with a strong focus on AI/ML and system-level engineering. I build custom Linux distributions, develop security and system tools, and work on machine learning applications.

My work focuses on creating practical software, building Linux distributions for different use cases, and applying AI/ML technologies to solve real-world problems.

Focus Areas

  • Software Development (Python, Go, Bash, JS)
  • Linux Distribution Development
  • AI/ML and Model Fine-Tuning
  • System-Level Engineering and Tooling
  • Linux Hardening and Blue Team

Current Work

Currently developing Zenclora OS (bloat-free Debian-based daily desktop with ZPM package manager), Linux Engineer Toolkit Live (specialized recovery and forensics distro), and Fishy Linux (lightweight distro for older hardware).

Also working on AI/ML projects including model fine-tuning, developing security tools with Python and Go, and exploring local LLM applications.

Mission

Building useful software and Linux distributions that empower users. I believe in open-source development, clean code, and giving users full control over their systems.

Every tool I build, every distribution I create, and every line of code I write serves this purpose: creating software that matters.
